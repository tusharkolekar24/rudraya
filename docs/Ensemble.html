get_weights(estimators, X_train, X_test, y_train, y_test, threshold=0.5)[source]

It assigne Weights to individual models of the estimators and Enhanced perforance of the Rank_Weighted_Regression.

Parameters

  • estimators (list of (str, estimator) tuples Invoking the fit method on the Rank_Weighted_Regression) – will fit clones of those original estimators that will be stored in the class attribute self.estimators_.

  • X_train ({array-like, sparse matrix} of shape (n_samples, n_features)) – The input samples. It is subset of main dataset which represent set of independent Features. It use for to train the model.

  • X_test ({array-like, sparse matrix} of shape (n_samples, n_features)) – The input samples. It is subset of main dataset which represent set of independent Features. It use for to test the model.

  • y_train ({array-like, sparse matrix} of shape (n_samples)) – The input samples. It is subset of main dataset which represent target Features. It use for to train the model.

  • y_test ({array-like, sparse matrix} of shape (n_samples)) – The input samples. It is subset of main dataset which represent target Features. It use for to test the model.

  • threshold (int or float) – default value is 0.5. Its range varies from 0 to 1. It allow model which accuracy > threshold value to form composite model.

Return type

Modified_weights, Modified_estimators
